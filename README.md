## Features

- Showcase my portfolio & skill excercises
- Can Post Blog using Ghost CMS backend
- Contact Form

## Tech Used

- HTML, CSS3 & JS
- Nextjs
- Reactjs
- Headless Ghost CMS backend (free for development purpose)
- Fontawesome icons
- Nodemailer + googleapi OAuth2
